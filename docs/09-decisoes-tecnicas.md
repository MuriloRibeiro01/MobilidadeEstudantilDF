# Decisões Técnicas

## DT01 — Uso de QR Code no MVP

Decisão: O MVP utilizará QR Code dinâmico em vez de NFC.

Motivo: NFC exige integração com sistemas de bilhetagem e acesso a chaves/protocolos proprietários.

## DT02 — Uso de React Native no frontend

Decisão: O Frontend será desenvolvido em React Native, utilizando TypeScript.

Motivo: Melhor integração com sistemas mobile, familiaridade e produtividade.

## DT03 — Uso de Laravel no backend

Decisão: O backend será desenvolvido em Laravel.

Motivo: Familiaridade, produtividade e facilidade para criar API, painel administrativo e autenticação.

## DT04 - Uso de PostgreSQL para database

Decisão: O banco de dados será desenvolvido em PostgreSQL.

Motivo: Familiaridade, escalabilidade e segurança.
